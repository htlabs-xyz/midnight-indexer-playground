# Tech Stack

## Runtime
- **Bun** - JavaScript runtime and bundler

## Server
- **Bun.serve()** - HTTP server with HTML imports

## Frontend
- **React 19** - UI framework
- **@graphiql/react** - Official GraphQL IDE components
- **graphql** - GraphQL parser/utilities
- **graphql-ws** - WebSocket subscriptions

## Styling
- **Tailwind CSS 4** - Utility-first CSS

## Key Dependencies
```json
{
  "react": "^19.0.0",
  "react-dom": "^19.0.0",
  "@graphiql/react": "latest",
  "graphql": "^16.9.0",
  "graphql-ws": "^6.0.4"
}
```

## Why This Stack?
1. **Bun** - Fast, modern runtime with native bundling
2. **GraphiQL** - Battle-tested GraphQL IDE components
3. **graphql-ws** - Standard subscription protocol
4. **Tailwind** - Rapid styling without leaving HTML
